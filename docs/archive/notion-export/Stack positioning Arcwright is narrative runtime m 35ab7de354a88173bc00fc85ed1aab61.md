# Stack positioning: Arcwright is narrative runtime middleware (Layer 2 in game dev stack)

Date: May 7, 2026
Rationale: Arcwright sits between game engines (Unity, Unreal, Godot, web) and AI providers (Anthropic, OpenAI, Groq). Engine handles rendering, input, audio. Arcwright handles arc orchestration, characters, knowledge state, content events. AI providers handle generation. Engine-agnostic and provider-agnostic. Closest analogue: http://Charisma.ai (more controllable, character-level vs session-level). Differentiated from: Convai (3D-native lock-in), Inworld (pivoted), Latitude/Voyage (consumer not platform), Ink/Articy (authored not generative).
Section: Cross-cutting
