# Progressive proprietary infrastructure: three-tier build sequence (deterministic now, fine-tuned models at scale, foundation models never)

Date: May 5, 2026
Rationale: Tier 1 at MVP: deterministic infrastructure with no AI cost (arc engine, session state, knowledge graph, routing, API). Tier 2 at Horizon 2 earliest, triggered by volume and cost data: fine-tuned open source models. Tier 3, never: foundation model development or compute ownership. Games are the data flywheel; session telemetry from day one is the mechanism. Competitors using the same APIs cannot replicate Arcwright-specific session data. Premature Tier 2 investment before data and economics justify it is a distraction from shipping.
