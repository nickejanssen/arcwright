# Which foundation model produces the best NPC character consistency for Nightcap's use case?

Category: Product
Date Opened: May 17, 2026
Priority: High
Status: Open

Emergence World's Season 1 ran identical environments with Claude Sonnet 4.6, Gemini 3 Flash, Grok 4.1 Fast, and GPT-5 Mini and found dramatically different behavioral outcomes across worlds. Model choice is therefore a narrative design decision, not just a cost or performance tradeoff. The same character with the same prompt may behave very differently depending on which model powers it. Arcwright should run a comparative test once one NPC is working in Nightcap: same mystery, same NPC profiles, different models, observe behavioral consistency and character fidelity. Must be resolved before committing the engine architecture to a primary provider.
