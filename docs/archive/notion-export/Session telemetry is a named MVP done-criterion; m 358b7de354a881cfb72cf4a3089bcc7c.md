# Session telemetry is a named MVP done-criterion; must be live from the first production session

Date: May 5, 2026
Rationale: Sessions without structured telemetry are compute cost with no data return. The Tier 2 fine-tuning transition is impossible without accumulated labeled session data. Minimum signals from first production session: arc beat engagement duration, pacing intervention triggers and outcomes, knowledge state constraint activations, session completion status, replay intent indicators. Richer schema designed in Chat 6.
