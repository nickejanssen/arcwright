# D-041 MVP scope pattern (schema-clean, implementation-staged)

Date: May 19, 2026
Rationale: Schema design is platform-clean from day one and includes all fields needed for full functionality. Implementation is staged. Nightcap MVP gets minimum-viable implementations of advanced features. Monster RPG H2 fills in sophisticated implementations. Schema does not change between MVP and H2. This synthesizes platform-clean and closer-to-Lighthouse: schema-clean preserves platform discipline, implementation-narrow respects solo-founder bandwidth.
