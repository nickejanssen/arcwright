# Session duration is emergent (30-75 min target range). No separate speed round mode.

Date: May 15, 2026
Rationale: Speed round is a fast group plus responsive pacing engine, not a designed configuration. Duration range replaces any session_mode parameter. The pacing engine manages within the 30-75 minute range based on player behavior. Both ends of the range are valid session outcomes.
