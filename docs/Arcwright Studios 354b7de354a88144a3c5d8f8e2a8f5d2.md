# Arcwright Studios

*Working name. Pending platform name resolution and entity formation.*

This is the project workspace for the company building the infrastructure for adaptive experiences.

## Mission

Our mission is to make adaptive experiences that move people. We do it by treating every player as a co-author, not an audience, so that imagination is the only ceiling.

## Quick reference

**One-line description:** [Company] is building the infrastructure for adaptive experiences: an API for stories, games, and simulations that respond to the people inside them, and the games that prove what becomes possible.

**Current horizon:** Horizon 1 (creative studio with internal infrastructure)

**Current wedge:** Adult social party games (Nightcap as flagship, working title pending trademark clearance)
