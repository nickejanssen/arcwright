# D-031 H2 external beta gate updated to require Nightcap profitability

Date: May 19, 2026
Rationale: H2 external developer beta is contingent on Nightcap profitability signal in addition to the existing five external developers building non-Nightcap experiences gate. This is a hybrid posture absorbing one of the closer-to-Lighthouse audit guardrails without abandoning the Pixar to Valve to Stripe progression. Supersedes prior H2 gate language in PRD Section 9.
