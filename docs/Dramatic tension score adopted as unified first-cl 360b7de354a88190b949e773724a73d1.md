# Dramatic tension score adopted as unified first-class session metric driving pacing, misdirection, and model tier selection simultaneously

Date: May 11, 2026
Rationale: A single composite float (0.0-1.0) computed from time pressure, player action rate, correct suspicion confidence, and clue coverage eliminates three separate threshold systems and creates a cleaner architecture. When tension peaks, the routing layer automatically upgrades character dialogue to premium tier without any explicit call-site code. Weights are arc-configurable in pacing_config, enabling iteration without code changes.
