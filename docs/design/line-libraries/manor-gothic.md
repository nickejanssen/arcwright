## Review Notes for the Founder

- This library leans hardest into **the house as co-narrator** ("the
  house declines," "the house keeps it now") — the gothic register's
  version of the diegetic-presence question (open question 1). If
  Vesper is an observer, the house can carry the memory instead of
  her; these lines are written so either answer survives with light
  edits.
- New slots: `{{room}}`, `{{weather}}`. The per-wrapper whitelist
  pattern holds.
- st-d-8 and st-o-8 give Vesper *forty years of service and a prior
  bad evening* — the strongest diegetic-history claims in any library;
  they are the probe for how much backstory the host may accrue.
- wa-6 (the topiary) and wa-7 (mislaid four seconds) are this
  wrapper's calibration probes.
- Lush-gothic was your third-preference register: this library is
  where "lush" lives in the current set. If it reads as the *right*
  amount of gothic, Crimson Peak-adjacent instances are covered; if
  you want lusher still, say so and the next pass raises the velvet.
