## Review Notes for the Founder

- **This is the range-proof library.** With Sim Reunion, the set now
  spans Vesper's full recipe: Boardroom (most reduced) to Sim Reunion
  (maximum Caine), with the launch pair in between. If both extremes
  read as *the same character*, the-host.md's one-voice thesis is
  proven in text.
- The drop section is this wrapper's biggest bet: grief rendered as a
  cheerful system failing to stay cheerful (co-d-2's missing font,
  co-d-3's "a voice I was not shipped with"). If it reads as comedy
  undercutting the death rule, the fallback is co-d-6's register.
- wa-6 (the koi complaints department) and wa-7 (joy experienced
  incorrectly) are the calibration probes — deliberately the furthest
  any library goes.
- New slots: `{{tier}}`, `{{errata}}`. Whitelist pattern holds.
- Diegetic-presence note: here Vesper being *inside* the world is the
  wrapper's premise (she IS the sim's host), which suggests the answer
  to open question 1 may legitimately vary by wrapper — worth
  considering before flattening it globally.
